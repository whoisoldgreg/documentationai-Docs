## Payment Processing

Process payments securely across multiple methods including credit/debit cards, ACH transfers, and digital wallets like Apple Pay or Google Pay. Scriptmasters handles Level 2 and Level 3 data for optimal interchange rates.

<Tabs>
  <Tab title="Credit/Debit Cards" icon="credit-card">
    Integrate card payments using our REST API. Capture full card details or use tokenized payments for PCI compliance.
    
    <CodeGroup tabs="JavaScript,cURL">
      ```javascript
      const response = await fetch('https://api.example.com/v1/charges', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${YOUR_API_KEY}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          amount: 2999, // $29.99 in cents
          currency: 'usd',
          source: 'tok_visa_example'
        })
      });
      const charge = await response.json();
      ```
      ```bash
      curl https://api.example.com/v1/charges \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{
          "amount": 2999,
          "currency": "usd",
          "source": "tok_visa_example"
        }'
      ```
    </CodeGroup>
  </Tab>
  <Tab title="ACH/eCheck" icon="banknote">
    Handle bank transfers with automated verification.
    
    <ParamField path="account_number" param-type="string" required="true">
      Customer's bank account number.
    </ParamField>
    
    <ParamField path="routing_number" param-type="string" required="true">
      Bank's routing number (9 digits).
    </ParamField>
  </Tab>
  <Tab title="Digital Wallets" icon="smartphone">
    Support seamless one-click payments from stored methods.
  </Tab>
</Tabs>

## Chargeback Defense

Scriptmasters offers 24/7 fraud monitoring, AI-assisted dispute tools, and chargeback guarantees. Receive alerts for suspicious activity and automate responses with pre-filled templates.

<Expandable title="Advanced Fraud Rules" default-open="false">
  Configure custom rules based on velocity, amount, or IP location.
  
  ```javascript
  const rule = {
    name: "High Velocity Check",
    conditions: [
      { type: "count", period: "1h", threshold: 5 }
    ],
    action: "block"
  };
  ```
</Expandable>
